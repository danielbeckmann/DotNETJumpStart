## Einrichtung

Klonen Sie das Repository oder laden es als ZIP-Archiv herunter. Wenn Sie das ZIP-Archiv heruntergeladen haben, müssen Sie es vor dem Entpacken "Zulassen". Hierzu machen Sie einen Rechtsklick auf das ZIP-Archiv und wählen **Eigenschaften**. Im unteren Teil des Fensters wählen Sie **Zulassen** aus und klicken OK.
