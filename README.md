## Einrichtung

Folgen sie diesen Schritten, um Ihren Computer für den Workshop vorzubereiten:

1. Installieren Sie [Virtual Box](https://www.virtualbox.org/).

2. Starten Sie das Image in Virtual Box, dass Sie zu Beginn des Workshops erhalten haben. Dieses enthält eine Evaluierungsversion von Windows 10 mit den bereits vorinstallierten Entwicklungstools.

3. Klonen Sie das Repository oder laden es als ZIP-Archiv herunter. [TODO] If you download as zip, remember to unblock the zip archive before unzipping. To unblock a downloaded archive, right-click on the zip in Windows Explorer, click Properties, then check __Unblock__ near the bottom of the window, then click __OK__.
